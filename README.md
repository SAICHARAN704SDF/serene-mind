WHAT MY WEBSITE PROVIDES (Features)
MY website is a Mental Wellness + EMR System, offering:
1. Mental Wellness Tools
AI chatbot (Google Gemini)
Mood journaling (add, edit, delete)
Mood tracking
Breathing exercise guide
Music therapy (songs list)
2. EMR (Electronic Medical Records)
Patient management
Doctor management
Appointment scheduling
Billing creation + editing
Payment status tracking
3. Admin Dashboard
Overview of all modules
Navigation to wellness + EMR sections
4. Cloud Deployed Application
Hosted on Render
Connected to managed PostgreSQL
Fully working CRUD operations
